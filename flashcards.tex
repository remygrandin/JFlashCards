The bellow set of 213 flashcards is set to help memorise Japanese kana.

They are divided in the following order : 

\begin{itemize}
  \item Hiragana
  \item Katakana
\end{itemize}

Then for each category :

\begin{itemize}
  \item Gojūon
  \item Dakuten
  \item Handakuten
  \item Yōon Dakuten
  \item Yōon Handakuten
\end{itemize}
